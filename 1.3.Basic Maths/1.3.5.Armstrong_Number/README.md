## Problem Statement  
Given a number x, determine whether the given number is Armstrong's number or not.
<br>
[Armstrong Number](https://www.quora.com/What-is-an-Armstrong-number) :An Armstrong number (also known as a narcissistic number) is a number that is equal to the sum of its own digits each raised to the power of the number of digits.
## Examples     <p>a<sup>3</sup> + b<sup>3</sup> + c<sup>3</sup> = abc</p>
<h2>3-digit Example: 153</h2>
        <p>
            1<sup>3</sup> + 5<sup>3</sup> + 3<sup>3</sup> = 1 + 125 + 27 = <strong>153</strong>
        </p>
<h2>4-digit Format: abcd</h2>
        <p>
            a<sup>4</sup> + b<sup>4</sup> + c<sup>4</sup> + d<sup>4</sup> = abcd
        </p>
        <p>Examples include:</p>
        <ul>
            <li>1634 → 1<sup>4</sup> + 6<sup>4</sup> + 3<sup>4</sup> + 4<sup>4</sup> = 1634</li>
            <li>8208 → 8<sup>4</sup> + 2<sup>4</sup> + 0<sup>4</sup> + 8<sup>4</sup> = 8208</li>
            <li>9474 → 9<sup>4</sup> + 4<sup>4</sup> + 7<sup>4</sup> + 4<sup>4</sup> = 9474</li>
        </ul>
