## Problem Statement 
 Write a program to reverse digits of a number.

## Examples
Input: n = 221
Output: 122
Explanation: By reversing the digits of number, number will change into 122.

Input: n = 200
Output: 002 ~ 2

Input: n = 345 
Output: 543
