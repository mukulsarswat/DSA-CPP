## Problem Statement 
Given a positive integer n, the task is to find the nth [Fibonacci number](https://en.wikipedia.org/wiki/Fibonacci_sequence).
<h3>F(n)=F(n-1)+F(n-2).</h3>

## Examples
Input: n = 3
Output: 2
Explanation: F(3) = F(2) + F(1) = 1 + 1 = 2.
Example 3:

Input: n = 4
Output: 3
Explanation: F(4) = F(3) + F(2) = 2 + 1 = 3.
